Chapter 9 · Number Systems

Rational Numbers

Summary

  • A rational number is any number that can be written as p/q, where q ≠ 0 (e.g., 3/4, −2/5, 7/1 are all rational numbers).
  • Every integer is a rational number (e.g., 5 = 5/1, −3 = −3/1, 0 = 0/1).
  • Positive rational numbers: both p and q have the same sign (e.g., 3/5 and −2/−7 = 2/7 are positive).
  • Negative rational numbers: p and q have different signs (e.g., −3/4 and 5/−6 are negative).
  • Equivalent rational numbers: multiply or divide numerator and denominator by the same non-zero number (e.g., 1/3 = 2/6 = 3/9).
  • Standard form: denominator is positive and there are no common factors between numerator and denominator (e.g., −4/6 in standard form is −2/3).
  • Rational numbers can be shown on a number line, just like integers (e.g., 1/2 is halfway between 0 and 1).
  • To add or subtract rational numbers, find a common denominator first (e.g., −1/3 + 2/3 = 1/3).

Key Formulas

Equivalent: a/b = (a × c) / (b × c) for any c ≠ 0
Addition: a/b + c/d = (ad + bc) / bd
Subtraction: a/b − c/d = (ad − bc) / bd
Multiplication: a/b × c/d = ac / bd
Division: a/b ÷ c/d = a/b × d/c

Worked Examples

Example 1: Identifying Rational Numbers

Is −3/5 a rational number?

Step 1: It is in the form p/q where p = −3 and q = 5
Step 2: q ≠ 0, so yes, it is a rational number

Example 2: Finding Equivalent Rational Numbers

Find an equivalent rational number of 2/3 with denominator 15.

Step 1: We need to multiply denominator by 5 to get 15
Step 2: Multiply both by 5: (2 × 5) / (3 × 5) = 10/15

Example 3: Adding Rational Numbers

Add −1/3 + 3/4

Step 1: Find common denominator: LCM of 3 and 4 = 12
Step 2: −1/3 = −4/12 and 3/4 = 9/12
Step 3: −4/12 + 9/12 = 5/12

Practice Problems

Easy

Write 7 as a rational number.

Show Answer
7 = 7/1
Easy

Is 0 a rational number?

Show Answer
Yes, because 0 = 0/1 (p=0, q=1, and q ≠ 0).
Medium

Write −6/8 in standard form.

Show Answer
Divide both by 2: −6/8 = −3/4 (denominator positive, no common factors).
Medium

Subtract: 3/5 − 1/3

Show Answer
LCM of 5 and 3 = 15. 3/5 = 9/15, 1/3 = 5/15. 9/15 − 5/15 = 4/15
Hard

Find: −2/3 × 9/4

Show Answer
(−2 × 9) / (3 × 4) = −18/12 = −3/2
Hard

Divide: 5/6 ÷ (−2/3)

Show Answer
5/6 × (−3/2) = −15/12 = −5/4
Easy

Is −5/0 a rational number?

Show Answer
No. A rational number is of the form p/q where q ≠ 0. Here the denominator is 0, so −5/0 is not a rational number.
Easy

Find the equivalent rational number of 3/4 with denominator 20.

Show Answer
Multiply numerator and denominator by 5: (3 × 5) / (4 × 5) = 15/20
Medium

Arrange in ascending order: −2/3, 1/2, −1/6, 3/4

Show Answer
LCM of 3, 2, 6, 4 = 12. Converting: −2/3 = −8/12, 1/2 = 6/12, −1/6 = −2/12, 3/4 = 9/12. Ascending order: −2/3, −1/6, 1/2, 3/4 (i.e., −8/12 < −2/12 < 6/12 < 9/12)
Medium

Add: −3/7 + 5/14

Show Answer
LCM of 7 and 14 = 14. −3/7 = −6/14. So −6/14 + 5/14 = −1/14
Hard

Simplify: (2/3 × −9/4) + (1/2)

Show Answer
First: 2/3 × −9/4 = −18/12 = −3/2. Then: −3/2 + 1/2 = (−3 + 1)/2 = −2/2 = −1
Hard

Find three rational numbers between 1/4 and 1/2.

Show Answer
Convert to equivalent fractions with a larger denominator: 1/4 = 4/16 and 1/2 = 8/16. Three rational numbers between them: 5/16, 6/16 (= 3/8), and 7/16

Common Mistakes

  • Forgetting that the denominator (q) can NEVER be zero.
  • Not finding a common denominator before adding or subtracting.
  • Writing standard form with a negative denominator — the denominator must be positive.
  • Forgetting to simplify the final answer to lowest terms.
