Chapter 12 · Algebra

Algebraic Expressions

Summary

  • An algebraic expression is made up of variables, constants, and operations (+, −, ×).
  • A term is a single part of an expression (e.g., in 3x + 5y − 2, the terms are 3x, 5y, and −2).
  • The coefficient is the number part of a term (in 3x, the coefficient is 3).
  • Like terms have the same variable raised to the same power (e.g., 3x and 5x are like terms).
  • Unlike terms have different variables or powers (e.g., 3x and 3y are unlike terms).
  • Monomial = 1 term, Binomial = 2 terms, Trinomial = 3 terms.
  • To add/subtract expressions: combine like terms.
  • To find the value of an expression: substitute the given values for the variables.

Key Formulas

Adding like terms: ax + bx = (a + b)x
Subtracting: ax − bx = (a − b)x
To find value: substitute the number for each variable and simplify

Worked Examples

Example 1: Identifying Terms and Coefficients

In the expression 4x² + 3xy − 7, identify the terms and coefficients.

Terms: 4x², 3xy, −7
Coefficients: 4 (for x²), 3 (for xy), −7 is a constant term

Example 2: Adding Expressions

Add: (3x + 5y − 2) + (2x − 3y + 7)

Step 1: Group like terms: (3x + 2x) + (5y − 3y) + (−2 + 7)
Step 2: Combine: 5x + 2y + 5
Answer: 5x + 2y + 5

Example 3: Finding the Value

Find the value of 2x² + 3x − 1 when x = 3.

Step 1: Substitute x = 3: 2(3)² + 3(3) − 1
Step 2: = 2(9) + 9 − 1 = 18 + 9 − 1
Answer: = 26

Practice Problems

Easy

How many terms are in the expression 5x + 3y − 2? Is it a monomial, binomial, or trinomial?

Show Answer
3 terms (5x, 3y, −2). It is a trinomial.
Easy

What is the coefficient of x in the expression 7x − 4y + 9?

Show Answer
The coefficient of x is 7.
Medium

Add: (4a + 3b) + (2a − 5b)

Show Answer
(4a + 2a) + (3b − 5b) = 6a − 2b
Medium

Subtract: (7x + 3) − (4x − 2)

Show Answer
7x + 3 − 4x + 2 = 3x + 5
Hard

Find the value of 3a² − 2ab + b when a = 2 and b = −1.

Show Answer
3(4) − 2(2)(−1) + (−1) = 12 + 4 − 1 = 15
Hard

Simplify: (5x² + 3x − 7) + (2x² − 4x + 3) − (x² + 2x − 1)

Show Answer
(5 + 2 − 1)x² + (3 − 4 − 2)x + (−7 + 3 + 1) = 6x² − 3x − 3
Easy

Identify the like terms in: 5x, 3y, −2x, 7, 4y

Show Answer
Like terms: 5x and −2x (both have variable x), 3y and 4y (both have variable y). The constant 7 has no like term.
Easy

Write the coefficient of y in: 8x + 3y − 12

Show Answer
The coefficient of y is 3.
Medium

Subtract (3a + 4b − 2) from (7a − 2b + 5)

Show Answer
(7a − 2b + 5) − (3a + 4b − 2) = 7a − 2b + 5 − 3a − 4b + 2 = 4a − 6b + 7
Medium

Find the value of 5x − 3y when x = 2 and y = −1

Show Answer
5(2) − 3(−1) = 10 + 3 = 13
Hard

Add: (2x² + 3xy − y²) + (x² − 5xy + 3y²) + (−x² + xy − y²)

Show Answer
(2 + 1 − 1)x² + (3 − 5 + 1)xy + (−1 + 3 − 1)y² = 2x² − xy + y²
Hard

If P = 3a² − 5a + 7 and Q = a² + 2a − 3, find 2P − Q.

Show Answer
2P = 6a² − 10a + 14. Then 2P − Q = (6a² − 10a + 14) − (a² + 2a − 3) = 6a² − 10a + 14 − a² − 2a + 3 = 5a² − 12a + 17

Common Mistakes

  • Adding unlike terms (e.g., writing 3x + 2y = 5xy — you can't combine them!).
  • Forgetting to change the sign of every term when subtracting an expression.
  • Mixing up coefficient and constant — the constant has no variable attached.
  • Forgetting the order of operations (BODMAS) when substituting values.
